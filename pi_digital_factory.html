    • Industry 4.0 concept

  • 4차 산업혁명의 모습 : 정보와 실체가 구조체계가 아닌 Network 방식으로 초 연결된 사회의 제조 측면에서의 단면, Connected World - Smart Factory - Smart Product, e-SCM 제조 혁신을 위한 디지털 신기술의 융합, CPPS 구현과 데이터 분석에 인공지능 활용 (Schwab 기술융합)
  • Enablers : IIoT, Big Data, Advanced analytics, Cloud, Advanced Robot, CPPS, 3D Printing, API, Mobile, 5G, Cognitive Computing, SCP, AR/VR/MR
  • Reference Model : 독일 RAMI 4.0, 미국 IIRA, 한국 스마트 공장 Architecture
  • 8대 스마트 제조기술과 스마트 공장운영 시스템 모델 & Case-Study :
  • CPPS - 사람, 기계장치, 운영 환경의 물리적인 환경과 Cyber 시스템을 연결하여 최적화하고 운영 제어가 되도록 한다. 제조의 한 부분을 넘어서 고객에게 제품/서비스를 전달하는 모든 과정에서 필요한 내용들의 초연결 (Hyper Connected Management)이며, 그것을 위한 제조기업의 정보관리 시스템(ERP, PLM, MES, CRM, SCM), 설비(PLC, CAD/CAM, Sensor, AI)와 공정 및 사람을 연결한 정보관리 체계이다.
  • 표준 참조 : IEC TR 62832-1 (Digital Factory Framework), IEC 62541 OPC-UA, IoT Gateway, Automation ML (Engineering tool's Data Integration )

  • CLOUD : 프로세스 중에서 무엇을 차별화 하여 특화 할 것인지를 정하고 남들과 비슷하거나 유사한 것은, 외부 CLOUD를 활용하여 표준 프로세스를 활용하도록 하고, 내부의 차별화가 필요한 것도 담기는 내용에 집중하고 처리하는 방식은 가능한 CLOUD로 처리하여 유연성을 높인다.
  • Open source : 많은 사람들이 참여하여 지속적으로 update되는 솔루션의 활용

  • 직원들의 디지털 기술 사용에 대한 자체 역량 강화를 위한 부분이 가장 중요하다 :

  • 새로운 사업을 할 경우에 시스템의 대응 속도와 지원 수준 고려 :
  • 새로운 조직으로 변화가 발생하거나 새로운 기계와 설비 도입에 따른 작업 방법의 변화에 프로세스와 시스템의 대응 고려 :
